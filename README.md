# Evaluación final módulo 1: Maquetación

Página web responsive desarrollada de acuerdo a un diseño dado. Para su creación se ha utilizado:

1. Sass.
2. Flexbox y CSS Grid.
3. Media queries.
4. Interacciones usando transiciones.

### Pre-requisitos

Este Kit incluye un motor de plantillas HTML, el preprocesador SASS y un servidor local.

Necesitas tener instalado [Node JS](https://nodejs.org/)

### Instalación

_Clona este repositorio_

```
git clone
```

_Instala los paquetes y dependencias de NPM_

```
git install
```

_Arranca el proyecto en el servidor local_

```
npm start
```

### Estructura de carpetas

```
src
 ├─ html
 |  └─ partials
 ├─ images
 ├─ js
 └─ scss
    ├─ core
    ├─ layout

```

Proyecto creado en **node y gulp**.
